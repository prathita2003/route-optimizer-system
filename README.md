# Route Optimization System

A full-stack application that computes shortest paths between nodes using graph algorithms.

## 🚀 Features
- Add nodes with coordinates
- Add weighted edges
- Compute shortest path
- Compare Dijkstra vs A* algorithm performance

## 🧠 Algorithms Used
- Dijkstra’s Algorithm
- A* Algorithm

## 🛠 Tech Stack
- Java (Spring Boot)
- Angular
- REST APIs

## 📊 Example Output
Path: A → C → D → F  
Distance: 11
